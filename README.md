# Simple_Todo_App
This is a simple todo app that makes use of react js in the client side  and nodejs as the server side 

---

## Features
1. CRUD operation
2. Authentication
3. Authorization
4. Personalise User todo
5. User profile
6. User signin and signout

---

## Tools
1. NodeJs
2. MongoDB
3. Express
4. ReactJs with Typescript
5. PassportJs
6. JsonWebToken(Jwt)
7. Cookie-Parser
8. Tailwindcss
9. Lucide-React
10. React-Icons
